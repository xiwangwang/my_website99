Brief Intro to S&P 500

First we import data “managers”, and extract monthly return of S&P 500. We have in total 132 observations ranging from 1996/1/31 to 2006/12/31, the mean is 0.00867, and the standard deviation is 0.0433.

data("managers")
sp500_return <- managers$`SP500 TR`
skim(sp500_return)

Now let’s plot the monthly returns for S&P 500.

plot(sp500_return * 100,
     main="S&P 500: 1-month % returns",
     ylab="return", xlab="date")

To build the index, we calculate cumulative returns for S&P 500 and draw a plot accordingly.

sp500_index <- cumprod(1 + sp500_return) #Cumulative Sums, Products, and Extremes
plot(sp500_index,
     main="S&P 500 Index",
     ylab="price", xlab="date")
